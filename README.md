# SSTAM-SSTAM

## Init

```
$ git clone https://github.com/Hate-Monday/sstam-sstam.git
$ cd ./sstam-sstam/react-native-client
$ npm i
```

## Pod Install

```
$ cd ./ios
$ pod install
```
